.phdconf
========

<br>

`cd ~`

`git clone https://github.com/phd/.phdconf`

`cd .phdconf`

`make home` (default),<br>
`make root`,<br>
`make etc`,<br>
`make apt`,<br>
`make apt-keys`,<br>
`make alternatives`,<br>
`make secure-boot`,<br>
`make niced`,<br>
or `make all` to install everything
